BD Bulk SMS API Manual - BDBulkSMS.net

Complete API Reference for Sending and Managing SMS

Introduction

API Links:

  • HTML Output: https://api.bdbulksms.net/api.php
  • JSON Output: https://api.bdbulksms.net/api.php?json

Non-SSL Version: You can also use http instead of https.

Generate API Token

Get your token here: https://gwb.li/token

Parameters: message, to, token

Request Method: POST or GET

Other API Usages (Balance, Expiry, Stats, etc.)

  • Plain Text Output: https://api.bdbulksms.net/g_api.php
  • JSON Output: https://api.bdbulksms.net/g_api.php?json

Examples

  • Check Balance: https://api.bdbulksms.net/g_api.php?token=yourtokencodehere&balance
  • Check Balance (JSON): https://api.bdbulksms.net/g_api.php?token=yourtokencodehere&balance&json
  • Check SMS Rate: https://api.bdbulksms.net/g_api.php?token=yourtokencodehere&rate
  • Total SMS Sent by Token: https://api.bdbulksms.net/g_api.php?token=yourtokencodehere&tokensms
  • Total SMS Sent by Main Account: https://api.bdbulksms.net/g_api.php?token=yourtokencodehere&totalsms
  • Current Month SMS from Main Account: https://api.bdbulksms.net/g_api.php?token=yourtokencodehere&monthlysms
  • Current Month SMS from Token: https://api.bdbulksms.net/g_api.php?token=yourtokencodehere&tokenmonthlysms
  • SMS Sent in Specific Month (Main Account): https://api.bdbulksms.net/g_api.php?token=yourtokencodehere&monthlysms=08-2025
  • SMS Sent in Specific Month (Token): https://api.bdbulksms.net/g_api.php?token=yourtokencodehere&tokenmonthlysms=08-2025
  • Check Expiry Date: https://api.bdbulksms.net/g_api.php?token=yourtokencodehere&expiry
  • All Data Together: https://api.bdbulksms.net/g_api.php?token=yourtokencodehere&expiry&rate&tokensms&totalsms&monthlysms&tokenmonthlysms
  • All Data Together (JSON): https://api.bdbulksms.net/g_api.php?token=yourtokencodehere&expiry&rate&tokensms&totalsms&monthlysms&tokenmonthlysms&json
